- Spring Boot
- Spring Data JPA
- Postgres sql

## How to Run

* git clone https://github.com/Abdalslam0aj/ClusteredDataWarehouseTask.git

* cd ClusteredDataWarehouseTask

* mvn clean install

* mvn spring-boot:run
